# 安装
- 克隆改仓库
- 进入根目录
- 运行命令: `npm install` 安装依赖

# 开发
- 运行开发服务器: `npm run serve`
- 服务器地址: http:localhost:3000
- Swagger文档地址: http:loaclhost:3000/doc
